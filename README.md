##Dependencies
- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

##Getting Started
- install all dependencies (using `npm install` command).
- Run the development web server using the `node express-server.js`
- Using a web browser navigate to http://localhost:8080/urls
